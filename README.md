## 📝 文档规范

1. **命名规则**: `项目名_文档类型_版本号.扩展名`
2. **版本管理**: 最新版在对应文件夹，旧版放入"旧版本归档"
3. **AI对话**: 按角色分类存放，不合并保留原始记录
4. **迭代记录**: 每次大版本更新需在归档中注明变更点
